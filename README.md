## Run the requirements.txt file to install the packages needed and then run main.py


Introducing Envio, the AI event and package manager that's here to save you from package management headaches. Instead of spending hours trying to figure out why one package won't work with another, let Envio handle it for you. It’s like having an AI agent that gets all your dependencies sorted without ever needing a break—just don’t forget to give it enough RAM and maybe a GPU. So go ahead, take a break, and let Envio do the hard work… until it decides to install everything under the sun. You're welcome!
